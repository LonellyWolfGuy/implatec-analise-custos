# Implatec — Análise de Custos

Comparativo de inventário entre dois meses a partir de relatórios MOD7 (PDF) exportados do TOTVS ERP.

## Como usar

```bash
cd app
npm install
npm run dev
```

Abra o navegador, faça upload de dois arquivos (Mês 1 e Mês 2) e visualize o comparativo.

## Formatos suportados

- PDF (MOD7 do TOTVS)
- CSV

## Stack

React 19 · TypeScript · Vite · Tailwind CSS 4
